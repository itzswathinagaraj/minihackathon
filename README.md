# 🎯 Event Management System — Mini Hackathon Project

> A web application to manage events — built with Python and Flask 
> as part of a college mini hackathon.

---

## ✨ Features

- 🔐 **Login & Register** — User authentication system
- 📅 **Create Events** — Admin can create and manage events
- 📋 **Event Details** — View full details of any event
- 👤 **My Registrations** — Users can track their registered events
- 🛠️ **Admin Dashboard** — Manage all events from one place

---

## 🛠️ Tech Stack

| Layer | Technology |
|---|---|
| Backend | Python, Flask |
| Frontend | HTML, CSS |
| Routing | Flask Blueprints |

---

## 🚀 Getting Started

```bash
git clone https://github.com/itzswathinagaraj/minihackathon.git
cd minihackathon
pip install -r requirements.txt
python app.py
```
